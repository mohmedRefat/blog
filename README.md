# Blog Application (React + Vite)

A modern **Blog Web Application** built using **React** and **Vite**.
The project focuses on performance, clean UI structure, authentication, and responsive design.

---

## Overview

This project is a frontend blog platform that allows users to interact with blog content through a modern React interface.
It integrates authentication and communicates with backend APIs to manage blog data.

The application is designed with scalability, maintainability, and responsiveness in mind.

---

## Tech Stack

* **React** – UI library for building component-based interfaces
* **Vite** – Fast development server and build tool
* **Clerk** – Authentication and user management
* **REST API** – Backend communication for blog data
* **CSS / Responsive Design** – Mobile-first responsive layout

---

## Features

* User authentication using **Clerk**
* Secure login and signup flow
* Blog posts display
* API integration for fetching blog content
* Component-based architecture
* Responsive design for mobile, tablet, and desktop
* Fast development with **Vite HMR**

---

## Authentication

Authentication is handled using **Clerk**, which provides:

* Secure sign-up and sign-in
* Session management
* User profile handling
* Protected routes

---

## API Integration

The application communicates with backend services through **REST APIs** to:

* Fetch blog posts
* Display post details
* Manage user-related operations

---

## Getting Started

### Install Dependencies

npm install

### Run Development Server

npm run dev

### Build for Production

npm run build

### Preview Production Build

npm run preview
