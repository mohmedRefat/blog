# React + Vite

This project is a **React application built with Vite**.  
Vite provides a fast development environment with instant server start and optimized production builds.

---

## Overview

This project uses **React** for building the user interface and **Vite** as the build tool and development server.  
Vite improves development speed by using native ES modules and providing fast Hot Module Replacement (HMR).

---

## Features

- Fast development server
- Hot Module Replacement (HMR)
- Optimized production builds
- Modern JavaScript support
- Clean and minimal project structure

---

## Tech Stack

- **React**
- **Vite**
- **JavaScript / JSX**
- **ESLint**
